# Dr. Frank Eduardo

Ecossistema digital dedicado à educação em saúde.

Este projeto reúne conteúdos, treinamentos, atendimento clínico e soluções tecnológicas para pacientes, fisioterapeutas, famílias e cuidadores.

## Objetivo

Construir a principal plataforma brasileira de educação em saúde voltada para dor, movimento, reabilitação e desenvolvimento profissional.

## Tecnologias

- Next.js
- TypeScript
- Tailwind CSS
- Vercel
